CentOS7安装neofetch

发表于2023-07-12|更新于2023-07-12|学习记录
|字数总计:142|阅读时长:1分钟|阅读量:

记录一下，万一以后忘了呢

CentOS7自带软件源内并无neofetch，我们需要手动添加软件源，

所有操作系统自带软件源内无此软件的，安装步骤大致为：添加软件源、导入密钥、刷新包管理器缓存、使用包管理器安装该软件

安装 epel-release

sudo yum install epel-release

添加第三方软件源

sudo curl -o /etc/yum.repos.d/konimex-neofetch-epel-7.repo https://copr.fedorainfracloud.org/coprs/konimex/neofetch/repo/epel-7/konimex-neofetch-epel-7.repo

使用包管理器安装 neofetch

sudo yum install neofetch

安装成功！
