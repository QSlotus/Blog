CentOS7安装neofetch
发表于2023-07-12|更新于2023-07-12|学习记录|CentOS| 条评论
记录一下，万一以后忘了呢 CentOS7自带软件源内并无neofetch，我们需要手动添加软件源， 所有操作系统自带软件源内无此软件的，安装步骤大致为：添加软件源、导入密钥、刷新包管理器缓存、使用包管理器安装该软件 安装 epel-releasesudo yum install epel-release 添加第三方软件源sudo curl -o /etc/yum.repos.d/konimex-neofetch-epel-7.repo https://copr.fedorainfracloud.org/coprs/konimex/neofetch/repo/epel-7/konimex-neofetch-epel-7.repo 使用包管理器安装 neofetchsudo yum install neofetch 安装成功！

致我的十五岁
发表于2023-01-21|更新于2023-01-21|日常|十五岁| 条评论
致我的十五岁。 第十五年。与这个世界交手的第十五年。感受世界万物美好的第十五年；品尝生活酸甜苦辣的第十五年；体验过喜怒哀乐惧的第十五年。第十五年，我曾奔波与医院与家中；第十五年，我曾枯坐于书桌前胡思乱想。第十五年，我有过惊慌失措，也有过无所畏惧；第十五年，我有过斗志昂扬，也有过划水躺平。我会因为在网上交到一个朋友而感到兴奋与激动，也会因许久没有联系的朋友的拉黑屏蔽而感到茫然与失落。我似乎与现实的隔阂越来越大。我仿佛对现实世界的一切都漠不关心。以前感到兴奋是常有的事，现在却是淡然，甚至对一切不理睬的冷漠。今日，除夕，大年三十。今日便会在我人生中第十五年画上一个句号，将过去整整一年的苦难与悲哀埋葬在记忆里面。第十五年。祝我自己生日快乐！希望新的一年越来越好！
