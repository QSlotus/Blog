重要的事情说三遍

项目介绍

叠甲完毕，进入正题。

此项目为某回合制二游的 免费开源 服务端实现，可在Github上输入神秘代码 Melledy/LunarCore 来查看其仓库 感谢Melledy大佬！
LunarCore 使用 Java 开发，最新提交适用于官方2.3.0客户端

现有功能有：

基本游戏功能：登录、队伍配置、背包、基本场景/实体管理
战斗功能
自然世界怪物/道具/NPC生成
大多数角色技能已处理
NPC商店已处理
祈愿系统
邮件系统
好友系统（好友支援尚未实现）
忘却之庭（带有1.4.0功能）
模拟宇宙（可以运行，但缺少许多功能）

来自 LunarCore README

开始运行

使用前请先至少连接到官方服务器一次以下载游戏数据

需要的东西

  • JDK 17
  • 项目源码及其编译好的核心
  • 资源文件
  • 网络代理工具（如 Cultivation ）

安装

  1. 安装好JDK17
  2. 将 Melledy/LunarCore 克隆到本地，打开Powershell，使用 ./gradlew jar 编译服务端核心
  3. 新建一个文件夹，将第二步编译好的核心（目录下的 LunarCore.jar）放入
  4. 在第三步新建的文件夹中再新建一个名为 resources 的文件夹以存放资源文件，并将 资源文件 解压至resources目录下
  5. 使用 java -jar LunarCore.jar 运行服务端
  6. 此时你会在服务端目录下发现一个名为 data 的子目录，然后将 LunarCore 仓库中的 ./data目录下的三个json文件复制到此目录下
  7. 再次运行服务端
  8. 打开 Cultivation，修改一些配置
    • 打开设置，将 Automatically Patch RSA | 自动修改RSA 的勾去掉
    • 在 Set Game Install Path | 选择游戏安装路径 选择你的游戏exe文件
    • 在工具主页填入你的 LunarCore HttpServer 的 IP 与 端口，记得勾选上 Connect to Grasscutter | 以 Grasscutter 模式连接
      • HttpSever的 ip 与 端口 可从 config.json 中的 httpServer.bindAddress 与 httpServer.bindPort 获取
    • 点击 Start | 启动
    • Cultivation 配置如下

      其他的保持默认即可
  9. Enjoy it!

账密相关

账号密码可以随便输，但是密码不会被保存。请保存好你的账号，它是你登入服务器的唯一凭据

获取分发服务器失败

  • 检查是否代理到了 HttpServer 的端口而不是 GameServer 的端口
  • 检查你的 hotfix url

安卓端怎么进
